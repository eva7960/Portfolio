AWS Sever less Image Pipeline (2025)

Time: 2 weeks Team Size: 3 Tools: AWS Lambda, S3 bucket, Ubuntu, FaaSRunner
We evaluated code generated by ChatGPT, Google Gemini, and Microsoft Copilot by having each LLM produce three Java-based AWS Lambda functions (rotate, resize, and grayscale) and deploying them in an image-processing pipeline. The pipelines were tested by retrieving images from an Amazon S3 bucket, processing them through the Lambda functions, and storing the results back in S3, while measuring code size (SLOC), latency, and throughput to compare performance across models. Overall we found ChatGPT provided the best user experience due to consistently lowest latency and round-trip times.
